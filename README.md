# Trendr

Trendr is a full-stack social media application developed using the MERN (MongoDB, Express.js, React.js, Node.js) stack. It allows users to create and interact with posts, connect with friends, and explore a social media experience.

## Project Demo Video
[Click here to watch the project demo video](https://drive.google.com/file/d/1cWvFBHCGvPEwVQcH9i4xAMxCppEzYeeg/view?usp=sharing)

## Technologies Used

- MongoDB
- Express.js
- React.js
- Node.js
- MaterialUI

## Features

- User authentication, registration, and login system using JWT tokens for secure access.
- Users can create, like, and interact with posts, fostering engagement within the community.
- Ability to add or remove friends, enabling users to connect with others on the platform.
- User profile pages showcasing user information and the posts they've made.

## Status

The Trendr project is currently ongoing, and development is in progress.

## Getting Started

To get started with the Trendr project locally, follow these steps:

1. Clone the repository: `git clone https://github.com/your-username/trendr.git`
2. Install dependencies: `npm install`
3. Set up environment variables. Create a `.env` file in the root directory and add the necessary variables.
4. Start the development server: `npm start`
5. Open your browser and navigate to `http://localhost:3000` to access the application.
